    ### 求大数字的对数时
    # 先用浮点近似，再用整数幂校正，避免大数下 log 精度导致的层级偏差
    n = int(math.log(l, 3)) if l > 0 else 0
    # 回摆修正：确保 3^n <= l < 3^(n+1)
    while pow(3, n + 1) <= l:
        n += 1
    while pow(3, n) > l:
        n -= 1
